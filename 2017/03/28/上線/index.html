上線

Mar 28, 2017 | Hits

今天本來也是抱著很平淡的心情去看看臉書的審查
沒想到竟然審核通過了
剛提交審查的後5分鐘就有人來測試

結果那時候我只有先架起來
根本沒寫指令
昨天把程式碼看一看之後補上指令
沒想到今天也有人來審查
一次就過了還真爽
讚！我的機器人上線了！！！！
真的超嗨的

雖然想做得智能一點
不過我知道這是非常難的一件事
要去分析語句
然後蒐集用戶的資料
現在跟我根本寫不出來
也沒什麼動力寫

既然上線了當然就要先搞出YR語錄
衝了

清醒有朋自遠方來
